Add An Organization to the List

This source for this website is open source, therefore anyone in the community submit edits through pull requests. If your agency isn't on this list but should be; please add it and submit a pull request.

  1. Log in to GitHub
  2. Go to this repository and fork it (button to the top right of the page).
  3. In your forked version, click on the _config.yml file.
  4. At the top of the file is a list of the organizations populating this page. Click edit, near the top right, and then add your organization's account name just like the others: - myagencysname.
  5. Click Commit at the bottom of the page.
  6. Visit the original repository and select Pull Request.
  7. Fill out the form and submit!
