Code, data and words are all simply text, so GitHub's collaboration features go way beyond just benefiting developers. permalink: /features/ ---

Collaborate

GitHub Issues lets you create discussion around text, proposed changes or just ideas. You can imbed images (A), ping other users or teams (B) and write in Markdown.

View

Visualizing data or text is lovely here. GitHub automatically creates maps (A) from .geoJSON files, tables (B) from .CSV, renders .markdown, shows the differences between saves of text (C) and images.

Share

GitHub makes it easy to share by generating urls for files, even for specific lines of text in a file (A). Forget emails and attachments, just ping another user and copy and paste a link -- we call that, the GitHub flow.

In your Datacenter or the Cloud

If you've got your own servers and want to run GitHub just within your agency, GitHub Enterprise has got you covered. If you don't want to worry about servers, we'll handle that for you and GitHub.com is your choice.

The Largest Open Source Network

GitHub is the word's largest open source network. With over 5 million users to learn from or get contributions from.
